mklog(pfx)
==========

Create a log file with prefix $pfx in the system temporary directory. Default 
prefix is 'log'. Filenames always contain the timestamp in '%Y%m%d%H%M%S' 
format.

log(msg)
========

Log a message to the logfile

plog()
======

Rread from pipe and log
